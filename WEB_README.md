# ממשק אינטרנט למילוי נוכחות אוטומטי

## איך להפעיל את השרת

1. **ודא שכל התלויות מותקנות:**
   ```bash
   pip install -r requirements.txt
   ```

2. **הפעל את השרת:**
   ```bash
   python web_server.py
   ```

3. **פתח דפדפן וגש לכתובת:**
   ```
   http://localhost:5000
   ```

## איך להשתמש

1. **העלה PDF** - גרור את קובץ דוח הנוכחות שלך או לחץ לבחירה
2. **הזן פרטים:**
   - מספר עובד
   - סיסמה
   - חודש יעד
3. **לחץ על "התחל מילוי אוטומטי"**
4. **המתן** - המערכת תציג התקדמות בזמן אמת
5. **הורד דוח ביקורת** - בסיום התהליך

## הערות חשובות

- ⚠️ **OTP**: כאשר המערכת תבקש קוד OTP, הזן אותו בחלון הדפדפן שנפתח
- 🔒 **אבטחה**: הסיסמה שלך לא נשמרת ונשלחת רק בזמן התהליך
- 💻 **מקומי**: השרת רץ על המחשב שלך ולא שומר מידע בענן
- 🌐 **שיתוף**: אם תרצי לשתף עם חברים, הם יכולים לגשת דרך הרשת המקומית שלך

## שיתוף עם חברים

אם תרצי שחברים יוכלו להשתמש במערכת:

1. **מצא את כתובת ה-IP שלך:**
   ```bash
   # Mac/Linux
   ifconfig | grep "inet "
   
   # Windows
   ipconfig
   ```

2. **שתף את הכתובת:**
   - החברים שלך יכולים לגשת ל: `http://YOUR_IP:5000`
   - לדוגמה: `http://192.168.1.100:5000`

3. **ודא שהשרת רץ:**
   - השאר את החלון של השרת פתוח
   - המחשב שלך צריך להיות מחובר לאותה רשת

## פתרון בעיות

### השרת לא עולה
- ודא שפורט 5000 לא תפוס
- בדוק שכל התלויות מותקנות

### הדפדפן לא נפתח
- זה תקין - המערכת משתמשת בדפדפן headless
- כאשר יש צורך ב-OTP, חלון ייפתח אוטומטית

### שגיאת התחברות
- ודא שמספר העובד והסיסמה נכונים
- בדוק את חיבור האינטרנט

## טיפים

- 📱 **OTP**: שמור את הטלפון קרוב כדי להזין את קוד ה-SMS במהירות
- 📄 **PDF**: ודא שהקובץ הוא דוח נוכחות תקין מהמערכת
- ⏱️ **זמן**: התהליך לוקח בדרך כלל 5-10 דקות תלוי במספר הימים
